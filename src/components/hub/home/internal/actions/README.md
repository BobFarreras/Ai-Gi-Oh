# Acciones de Home

## Qué contiene

1. Inserción de carta seleccionada.
2. Retirada de carta seleccionada.
3. Evolución de carta seleccionada.

## Objetivo

Mantener `HomeDeckBuilderScene` como orquestador de UI y mover flujos async/optimistas a módulos reutilizables.
