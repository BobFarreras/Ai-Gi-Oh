# Tutorial Flow Components

Componentes reutilizables para guías paso a paso:

1. `TutorialSpotlightOverlay`: foco visual del objetivo activo.
2. `TutorialInteractionGuard`: bloqueo de interacciones fuera del objetivo permitido.
3. `TutorialBigLogDialog`: diálogo narrativo y botón `Siguiente`.
4. `useTutorialFlowController`: orquestación local del estado del flujo.
